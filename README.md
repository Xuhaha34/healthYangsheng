# 四季养生食谱小程序

一款国风轻量化饮食养生科普小程序，基于 uni-app 开发，可编译为微信小程序、H5 等多个平台。

## 设计原则

- **简约为主**：无多余装饰，核心功能
- **统一国风主题色彩**：主色 #8C4A31（棕褐色国风原木色），辅助色 #F6EBE0（浅米杏底色）
- **轻量化、目标专一**：专注四季饮食养生科普，无商城、社交、广告等无关功能

## 页面结构

共 8 个页面：

| 页面 | 核心功能 |
|------|----------|
| index | 首页，轮播图、节气养生、快速入口 |
| seasonFood | 四季食谱，春夏秋冬分类 |
| ingredient | 食材科普，药食同源食材功效查询 |
| collect | 我的收藏，收藏的食谱与食材 |
| selfTest | 饮食自测，体质测试与建议 |
| setting | 设置，字体大小、数据管理 |
| detail | 详情，食谱/食材详情查看 |
| mine | **我的**，个人中心 + AI养生顾问 |

## 核心功能

### 基础功能
1. **首页自动轮播**：3 秒自动切换，展示春季养肝、夏季清热、秋冬滋补
2. **底部 Tab 切换**：首页、四季食谱、我的收藏、**我的**四个主 Tab
3. **食谱分类浏览**：春夏秋冬四季养生菜肴，含食材/步骤/功效
4. **食材科普查询**：10+ 药食同源食材，支持搜索和分类筛选
5. **一键收藏**：食谱和食材可收藏，本地持久化存储
6. **体质自测**：5 道简易题目，分析 6 种体质类型并给出饮食建议
7. **节气提示**：根据当前日期自动显示节气养生建议
8. **设置中心**：字体大小调节、数据统计、清除收藏

### 🌟 新增功能：我的页面 & AI养生顾问

**个人中心（mine）**
- 用户信息展示，含体质测试结果徽章
- 四格数据统计面板：食谱收藏、食材收藏、体质测试、AI咨询次数
- 功能菜单入口：快速跳转到自测、收藏、食材科普等页面
- 每日养生小贴士：每天展示不同的养生箴言
- 清空AI对话记录管理

**🤖 AI养生顾问**（智能对话功能）
- 自然语言对话界面，仿IM消息气泡设计
- 快捷问题一键发送：夏季养生、体寒调理、失眠、肠胃调理等
- 智能关键词识别：
  - **季节调理**：识别"春/夏/秋/冬/寒/热/燥"等，给出对应季节养生建议
  - **症状调理**：识别"失眠/肠胃/体寒/上火"等，提供针对性饮食方案
  - **食谱推荐**：识别"推荐/食谱/菜"，根据当前季节推荐养生菜肴
  - **食材查询**：识别"食材/功效/作用"，展示药食同源食材功效
  - **问候交互**：识别"你好/您好/hi"，友好引导用户提问
- 个性化响应：结合体质测试结果提供定制建议
- 本地持久化：对话记录保存到本地存储，下次打开可继续
- AI思考动画：模拟真实AI的延迟响应，提升交互体验
- 对话时间戳：每条消息显示发送时间

## 技术实现

- **框架**：uni-app + Vue 2
- **组件**：swiper、input、radio、button、view、text、image 等
- **样式**：rpx 自适应、flex 布局、统一主题色
- **数据**：本地 mock 数据 + uni.setStorageSync 持久化
- **事件**：@tap 点击事件、@input 输入事件、@change 切换事件
- **路由**：uni.navigateTo、uni.switchTab、页面传参
- **AI对话**：基于规则匹配的智能回复引擎 + 本地存储持久化

## 使用方法

1. 使用 HBuilderX 打开 `healthYangsheng` 目录
2. 选择「运行」→「运行到小程序模拟器」→「微信开发者工具」
3. 或选择「运行」→「运行到浏览器」查看 H5 版本
